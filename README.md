## Overview

**LemonPyHub** delivers polished, user-friendly software designed to improve system performance and simplify everyday tasks for Windows users.  
Each application is developed in **Python** with a graphical user interface (GUI) and compiled into executable (`.exe`) binaries to provide a zero-setup experience.  

**Supported platforms:** Windows 10 and Windows 11.

## Installation & Usage

1. Visit the [Download page](https://lemonpyhub.github.io/download/) or the **Releases** section on GitHub.  
2. Download the `.rar` package for the desired app.  
3. Extract the `.rar` (WinRAR, 7-Zip, or Windows builtin).  
4. Run the `.exe` file ‚Äî no installation, no Python environment needed.

The applications are portable and ready to use immediately after extraction.
